1. Headings (h1 to h6)

Headings are used to create a hierarchy of information. h1 is most important, h6 is least. Search engines and assistive technology rely on them.

This is Heading 1 (h1) - Main Title

This is Heading 2 (h2) - Section Title

This is Heading 3 (h3) - Subsection

This is Heading 4 (h4) - Minor heading

This is Heading 5 (h5) - Smaller heading
This is Heading 6 (h6) - Smallest heading

Use case: Structure your content logically. Start with H1, don't skip levels.

4. Font Families

Different fonts create different moods and readability levels in your website.

Arial (Sans-serif): Clean, modern, great for body text

Times New Roman (Serif): Traditional, newspaper-like appearance

Courier New (Monospace): For code snippets: if (x > 5) { return true; }

Georgia (Serif): Elegant and readable, great for blogs

Trebuchet MS (Sans-serif): Web-safe and friendly appearance

Best Practice: Combine fonts with fallbacks: font-family: 'Primary Font', 'Fallback Font', generic-family;

Quick Reference Summary

Concept HTML Element/CSS Purpose
Headings <h1> to <h6> Create content hierarchy
Bold <strong> or <b> Emphasize important text
Italics <em> or <i> Show emphasis or mention
Underline <u> Highlight or draw attention
Color style="color: [value]" Change text color
Background style="background-color: [value]" Add background color
Font style="font-family: [name]" Change text typeface
Font Size style="font-size: [value]" Adjust text size
Line Break <br> Break text into new line
Horizontal Line <hr> Visual section separator

Practical 5: HTML Forms

Learn to create interactive forms for collecting user data.

1. What are HTML Forms?

Concept: Forms are used to collect user input and send it to a server for processing.

Basic Form Structure

The <form> element wraps all form controls. Key attributes:

  • action - Where to send the data
  • method - How to send (GET or POST)
<form action="/submit" method="POST">
    <!-- form controls go here -->
</form>

Use Case: Login pages, registration forms, search boxes, contact forms.

8. Summary

Key Points:

  • <form> - Container for all form elements
  • <input> - Various types: text, email, password, radio, checkbox
  • <select> - Dropdown menus with <option>
  • <textarea> - Multi-line text input
  • <label> - Describes form controls (use for attribute)
  • <button> - Submit and reset actions
  • required - Makes a field mandatory

Best Practices:

  • Always use labels for accessibility
  • Group related radio buttons with the same name
  • Use appropriate input types for better mobile experience
  • Provide clear error messages
  • Include both submit and reset buttons

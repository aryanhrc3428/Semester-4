Practical 4: Frames in HTML

1. What are Frames?

Frames allow you to divide a browser window into multiple independent sections, each capable of displaying a different HTML document.

Note: The <frameset> tag is not supported in HTML5. Modern websites use CSS or <iframe> instead.

Basic Frame Structure (Deprecated)

<frameset cols="25%,75%">
    <frame src="menu.html" name="menu">
    <frame src="content.html" name="content">
</frameset>

2. Using iframes (HTML5 Standard)

The <iframe> tag embeds another HTML page within the current page.

Example: Basic iframe

Code:

<iframe src="https://example.com" width="400" height="200">
    Your browser does not support iframes.
</iframe>

Result:

Common iframe Attributes

Attribute Description Example
src URL of the page to embed src="page.html"
width Width of the iframe width="500"
height Height of the iframe height="300"
name Name for targeting name="content"
frameborder Border around iframe (0 or 1) frameborder="0"

4. Linking to Frames

You can make links open in specific frames using the target attribute.

Target Attribute Values

  • _self - Opens in same frame (default)
  • _blank - Opens in new window/tab
  • _parent - Opens in parent frame
  • _top - Opens in full window
  • framename - Opens in named frame

Example:

<a href="page.html" target="contentFrame">Open in Frame</a>
<iframe name="contentFrame" src="default.html"></iframe>

6. Summary

  • Frames divide browser window into multiple sections
  • <frameset> is deprecated in HTML5
  • Use <iframe> for embedding external content
  • Use CSS for creating frame-like layouts
  • target attribute controls where links open
Best Practice: For modern web development, prefer CSS layouts over frames for better accessibility, SEO, and mobile responsiveness.
